Tipo de drill
01 — Estrutura
1
Formato
Eliminatória direta com sorteio aleatório. O número de jogadores é arredondado para a próxima potência de 2 (2, 4, 8, 16…) com byes automáticos para os jogadores excedentes.
2
Marcação
Os 2 jogadores combinam o match entre si, quando quiserem. O coach faz um nudge periódico no grupo com os matches pendentes.
02 — Regras do match
1
Escolha dos drills
A escolha é feita por ordem alfabética — o jogador cujo nome vem primeiro no alfabeto escolhe o seu drill primeiro. O segundo jogador escolhe de seguida, mas não pode escolher o mesmo drill. Ambos comunicam a escolha ao adversário na hora do match.
2
Ordem de jogo
Quem começa cada drill é sempre o jogador que NÃO escolheu esse drill.

Exemplo: Jogador A escolheu Drill 1, Jogador B escolheu Drill 2.
→ No Drill 1, começa o Jogador B.
→ No Drill 2, começa o Jogador A.
3
Execução
Cada jogador faz 50 mãos no Drill 1 e 50 mãos no Drill 2. O score do GTO Wizard é registado para cada drill individualmente.
4
Resultado
Soma dos scores dos 2 drills. Maior total ganha e avança no bracket.
03 — Situações especiais
⚖️
Empate
Se os scores totais forem iguais, o coach escolhe 1 drill de desempate. Ambos os jogadores fazem o mesmo drill. Melhor score avança.
🚪
Desistência
Se um jogador desistir a qualquer momento, o adversário avança automaticamente (walkover).
Drills — todos os drills têm de ser cEV. Não é permitido escolher drills não-cEV para obter vantagem de score.
04 — Objetivo
Este torneio não é para ficar rico — é para criar um ambiente de prática competitiva que incentive o estudo e o drill consistente. Buy-in e prémios definidos pelo grupo antes do início.
